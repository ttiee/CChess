### 五子棋大作业

#### 1. 项目简介

本项目是一个简单的基于MFC的五子棋游戏，实现了五子棋的基本功能，包括人机对战、~~双人对战~~、悔棋、重新开始等功能。项目使用MFC框架，主要使用了MFC的视图类、对话框类、消息映射等技术。项目的主要难点在于实现五子棋的规则，以及实现人机对战的算法。项目的主要功能已经实现，但是还有一些功能有待完善，比如~~悔棋功能的优化~~、界面的美化等。
